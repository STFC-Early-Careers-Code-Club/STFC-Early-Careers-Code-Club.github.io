# STFC Early Careers Code Club

Welcome to the official site for STFC's Early Careers Code Club!

## What is this?

Code Club is a weekly meeting hosted at STFC in which early careers employees can learn from eachother.

The agenda for code club is as follows:

1) [Talk](/talks) - A presentation given by a speaker
2) Discussion - If anyone is struggling with a coding issue they can get some help on it
3) Coding Challenge - Small coding challenge completed by each member

## How do I join?

Code Club is open to all early careers employees at STFC. To join, simply turn up to thee weekly meeting in person!

These meetings are held every Wednesday at 12:30, usually in a B Block meeting room (DL).

If you cannot attend in person you can join via Teams. [Ashraf Hussain](mailto:ashraf.hussain@stfc.ac.uk) will send out a Teams link before each meeting.
