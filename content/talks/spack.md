However, here are a few useful links to get started with Spack:
- [Spack's main landing page](https://spack.io/)
- [Why to use Spack](https://spack.readthedocs.io/en/latest/features.html)
- [Spack's getting started guide for users](https://spack.readthedocs.io/en/latest/getting_started.html)
- [A live demo/tutorial of Spack from its creators (warning: quite long)](https://www.youtube.com/watch?v=Uoi3-_xMPtk)
- [A website where you can check all software already supported by Spack](https://packages.spack.io/)
- [A guide on how to make your own Spack packages (recipes for installation) for your own software/any software you use that isn't already supported](https://spack.readthedocs.io/en/latest/packaging_guide_creation.html)
